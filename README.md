# Online Bookstore SQL Project

## Overview
This project demonstrates database design and business analysis using PostgreSQL.

## Database Tables
- Books
- Customers
- Orders

## Skills Used
- SQL
- PostgreSQL
- Joins
- Aggregate Functions
- Group By
- Having
- Data Analysis

## Business Questions Solved
- Total Revenue Generated
- Most Frequently Ordered Book
- Top Spending Customer
- Books Sold by Genre
- Remaining Stock Analysis

## Dataset
The project uses:
- Books.csv
- Customers.csv
- Orders.csv
