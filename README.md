# Online Bookstore SQL Project

## Overview
This project is built using PostgreSQL and focuses on managing books, customers, and orders in an online bookstore. The project includes database creation, data import from CSV files, and SQL queries to perform business analysis.

## Database Tables

- Books
- Customers
- Orders


## Skills used

* SQL
* PostgreSQL
* Database Design
* Data Import using CSV Files
* Joins
* Aggregate Functions
* GROUP BY
* HAVING
* DISTINCT
* ORDER BY
* COALESCE
* Business Data Analysis

## Business Questions Solved

### Basic Analysis

* Retrieve books by genre
* Find books published after a specific year
* List customers from a particular country
* Identify orders within a date range
* Calculate total stock available
* Find the most expensive book
* Find the book with the lowest stock
* Calculate total revenue

### Advanced Analysis

* Total books sold by genre
* Average price of Fantasy books
* Customers with multiple orders
* Most frequently ordered book
* Top spending customer
* Total books sold by each author
* Cities with customers spending more than $30
* Remaining stock after fulfilling orders

## Dataset

The project uses the following datasets:

* Books.csv
* Customers.csv
* Orders.csv
